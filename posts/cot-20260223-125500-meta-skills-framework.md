## Core Thesis

- Downloaded skills ≠ real capability (artifact without reasoning)
- 3-layer meta-skill framework:
  1. **Trigger System** — When/why skill activates
  2. **Thinking Architecture** — How to reason about problem class
  3. **Verification Gate** — Self-audit against generic output

**Key Quote:** "Instructions produce results. Thinking architecture produces reasoning."

## 3 Approaches

### 1. Build Personal Meta-Skill Stack
Encode YOUR mental models, not downloaded ones. Document failures → extract patterns → formalize into 3 layers.

### 2. Reverse-Engineer & Rebuild
Dissect popular skills. Map to 3 layers. Identify weak triggers, missing thinking, no verification. Rebuild with your expertise.

### 3. Teach the Framework (Monetize)
Course creators: teach meta-thinking, not prompt templates. Differentiation + premium pricing.

## Advanced Concepts (Part 2)

### The Contrarian Frame
Before building ANY skill, answer: "What would the lazy version look like?"
- What structure would AI reach for first?
- What vocabulary would it default to?
- What assumptions would it bake in?

Then **engineer AWAY from each one**.

> "Baseline AI behavior is statistical averaging... if you don't carve out the negative space, output gravitates toward the center — and the center is where forgettable lives."

### Context Window Management
Your context window has 3 tenants fighting for room:
1. Instructions
2. Your input
3. Generation capacity

**Progressive Disclosure (3 Tiers):**
- **Tier 1 (Always-on):** Lean orchestrator <500 lines, routing logic only
- **Tier 2 (On-demand):** Domain knowledge, loaded when specific phase requires
- **Tier 3 (Verification):** Quality gate, loaded LAST (freshest in memory)

> "The main file is a router, not a textbook."

### Expert Panel Problem
❌ "Pretend to be Expert X" = cosplay, pattern-matching to what sounds smart
✅ Build **cognitive profiles** from real methodology:
- Recurring decision frameworks (not vocabulary)
- Prioritization logic (what they look at FIRST)
- Red flags (what makes them suspicious)
- Questions they ask before committing
